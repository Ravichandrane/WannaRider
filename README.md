## Dependencies :
### Front (Web Application)
	- AngularJS
	- NodeJS
	- Yeoman
	- Bower
	- Grunt

### Back (ReST API)
	- FatFreeFramework
	- JSON Web Token
	- MySQL

## APIs
	- Google Maps
	- Weather (Forecast)
	- API Wanna Ridde (Our own ReST API)

## To start front project

Clone and write command line :

```
    $ npm install
```

Then Enjoy ! ;)
